## Orbit Window Manager

Browser-native window management for tmux-backed terminals.

**Main Uses**
- Live dashboards for production logs on a shared screen.
- Multi-terminal dev workflows in a single browser window.
- Persistent workspaces that reopen exactly as you left them.
- Fast context switching between projects and environments.
- Lightweight collaboration with shared, read-only views.
- Centralized local infrastructure with tmux-backed reliability.
- Operations command centers for incidents and deploys.
- Future-ready remote access when the secure backend is enabled.
